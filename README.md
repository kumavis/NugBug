NugBug
======

Unity3D Editor Interface for NuGet Package Manager.

[  not fully functional at this time  ]


from the Unity editor you can:

[✗] Search

[✗] Pull

[✗] Publish

[✓] Use NuGet CLI


Prerequisites
=============

Mac, Linux - `mono`
PC - nothing!
